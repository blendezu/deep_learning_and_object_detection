# DLO - Deep Learning for Computer Vision

This project documents experiments and implementations for Deep Learning in Computer Vision using PyTorch and OpenCV.

## Requirements

- Python 3.8+
- PyTorch
- OpenCV
- NumPy

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the main script:
   ```bash
   python main.py
   ```
